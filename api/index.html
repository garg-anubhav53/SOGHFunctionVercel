SOGH Email Deploy API

Welcome to the SOGH Email Deploy API. This API provides endpoints to scrape email information from GitHub and Stack Overflow profiles.

Health Check

GET /health

Check if the API is running.

curl https://your-api-url/health

Scrape GitHub Profile

POST /scrape/github

Scrape email information from a GitHub profile.

curl -X POST https://your-api-url/scrape/github \
    -H "Content-Type: application/json" \
    -d '{"github_url": "github.com/username"}'

Scrape Stack Overflow Profile

POST /scrape/stackoverflow

Scrape GitHub email information from a Stack Overflow profile.

curl -X POST https://your-api-url/scrape/stackoverflow \
    -H "Content-Type: application/json" \
    -d '{"stackoverflow_url": "stackoverflow.com/users/user_id/username"}'
